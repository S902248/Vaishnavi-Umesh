# 💍 Wedding Invitation Text Animation 🎉💌

A **beautiful and elegant animated wedding invitation** built using **React, Vite, and Framer Motion**. This project combines **text animations**, **decorative effects**, and an interactive **scratch card feature** to create a fun and engaging wedding invitation for your guests.

---

## 💖 Features

### Text Animations
- **Fade-in / Slide-in Text** ✅ – Smooth entrance for couple names, date, and venue.
- **Typewriter Effect** ✅ – Text appears letter by letter for a classic invitation feel.
- **Gradient Text Animation** ✅ – Shimmering gold and rose gradients for extra elegance.
- **Sparkling / Glowing Names** ✅ – Letters softly sparkle with a romantic touch.
- **Curtain Reveal Animation** ✅ – Theatrical title reveal with sliding curtains.

### Decorative Animations
- **Floating Petals / Confetti** ✅ – Gentle falling rose petals in the background.
- **Sparkling Lights / Bokeh** ✅ – Dreamy, glowing background elements.
- **Heart Pulse Animation** ✅ – Small romantic hearts pulsing around the text.
- **Fireworks Animation** ✅ – Subtle sparkle bursts for celebratory effect.

### Interactive Feature
- **Scratch Card Effect** (Coming Soon) – Guests can “scratch” an overlay to reveal hidden invitation details like names, date, and venue.
  - Uses **canvas overlay** for a silver/gold scratch effect.
  - Works with **mouse and touch devices**.
  - Can trigger **celebratory effects** like confetti when fully revealed.

## 🛠️ Technologies Used

- **React** – Component-based UI logic.
- **Vite** – Lightning-fast build tool and dev server.
- **Tailwind CSS v4** – Modern utility-first styling.
- **Framer Motion** – Industry-leading animation library.
- **HTML5 Canvas** – Interactive scratch-off functionality.
